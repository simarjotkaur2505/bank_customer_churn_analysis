# Customer Churn Analysis using MySQL

## Project Overview
Analyzed customer churn data using SQL to identify churn patterns and customer retention insights.

## Tools Used
- MySQL
- SQL
- Kaggle Dataset

## Dataset Source
https://www.kaggle.com/datasets/shubh0799/churn-modelling

## Key Insights
- Germany had highest churn rate
- Inactive users churned more
- Older customers were more likely to exit
